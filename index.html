HTML

  • HTML is used to create structure of a web page
  • HTML is made up of elements which work together to provide content and basic framework for all pages on web
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

Most common type

  1. Text Elements
  2. Image Elements
  3. Header
  4. List
  5. Meta-related elements
  6. Content Divion
  7. Sematic Elements

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

GIT

GIT hub is a hosting service that stores data in the cloud. And GIT is a version control sytem . To access git, we use command line

GIT Commands

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to i
  • git add-A: stage a file
  • git init: initialize an existing repository
  • git clone(url): retrive an existing repository from a hosted location via URL
  • git commit -m: commit your staged file
